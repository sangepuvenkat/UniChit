 🧠 Problem

Students often struggle with:
- Paying college fees
- Buying laptops/courses
- Managing emergency expenses

Traditional options:
- ❌ Loans → high interest  
- ❌ Informal chit funds → risky & unstructured  

---

 💡 Solution

UniChit provides:
- ✔️ Secure digital chit system  
- ✔️ Transparent bidding mechanism  
- ✔️ Automated bonus distribution  
- ✔️ Student-focused financial model  

---

 ⚙️ Key Features

- 🔐 Authentication (Firebase Auth)
- 👥 Group creation & management
- 💰 Monthly bidding system
- 📊 Automated commission & bonus calculation
- 📅 Payment tracking system
- 🏫 College verification system
- 🤖 AI-based financial assistant (ChatGPT integration)

---

 🛠 Tech Stack

- **Frontend:** Next.js 14, TypeScript, Tailwind CSS  
- **Backend:** Firebase (Auth, Firestore, Storage)  
- **UI:** shadcn/ui  
- **Deployment:** Vercel  

---

 🧩 System Design

- Designed scalable Firestore schema for:
  - Users
  - Groups
  - Bids
  - Payments  

- Implemented:
  - Real-time bidding logic  
  - Role-based access (Leader / Student)  
  - Secure transaction flow  

---

 📊 How It Works

1. Join or create a chit group  
2. Contribute monthly amount  
3. Participate in bidding  
4. Winner receives lump sum  
5. Continue payments till cycle ends  

---

 🔥 What Makes It Unique

- Built specifically for **students**
- Fully **digital & automated**
- Transparent financial model
- Eliminates need for physical chit meetings
